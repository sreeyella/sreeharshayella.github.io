Brief summary of some html tags

All html doucments start with a statement <!DOCTYPE html> which specifies the browser to interpret the text below as HTML and display it accordingly. HTML uses different tags to present the content in an effective way. Most html tags contain a opening and a closing. Together with the content inbetween the opening and closing tags, the whole thing is called an html element. Below is a gist of some basic html tags.

  • <html>, </html>: This is the main tag in an html document. Everything else resides in between these tags.
